Introduction

jQuery.dropmenu is a plugin which takes a standard HTML-list (UL-element) and turns it into a dropdown-menu with (in theory an infinite amount of) submenu's. It is easy to install and style with CSS.
The plugin is designed to be quickly applied: Most of the configuration like heights, widths and distances are calculated automatically.




Examples

An UL-list with sub-UL-lists with the dropmenu-plugin and some basic CSS definitions:

$('#basismenu > ul').dropmenu({
	effect	: 'slide',
	nbsp	: true
});

Installation

Include the jQuery library, the dropmenu plugin and the dropmenu CSS inside the <head> tag of the page:

<script language="javascript" type="text/javascript" src="jquery.js"></script>
<script language="javascript" type="text/javascript" src="jquery.dropmenu.js"></script>
<link type="text/css" rel="stylesheet" href="dropmenu.css" />

Create a list with sub-lists and put the text of every item in an a- of span-element.

<ul id="foo">
	<li><span class="selected">TopItem 1</span></li>
	<li><span>TopItem 2</span>
		<ul>
			<li><a href="/">Subitem 2.1</a></li>
			<li><a href="/">Subitem 2.2</a></li>
			<li><a href="/">Subitem 2.3</a></li>
		</ul>
	</li>
	<li><a href="/">TopItem 3</a></li>
</ul>

Add the following JavaScript inside the <head> tag of the page:

<script language="javascript" type="text/javascript">
	$(document).ready(function() {
		$('#foo').dropmenu(
			{	/*	Configuration goes here	*/	}
		);
	});
</script>



Configuration

A list of the options to set (showing default settings):

$.fn.dropmenu.defaults = {
	effect			: "none",		//	"slide", "fade", or "none"
	speed			: "normal",	//	"normal", "fast", "slow", 100, 1000, etc
	timeout			: 250,
	nbsp			: false,
	maxWidth		: 0
}

effect String
The effect to animate the opening and closing of the submenu's.

speed Number
The duration of the animation in milliseconds, can also be one of the by jQuery predefined speeds like "fast" or "slow".

timeout Number
The amount of milliseconds before a subitem will close after mouse-out.

nbsp Boolean
If the text of the items need to stay on one line. This is desirable when the items do not have a predefined with and/or do have a predefined height.

maxWidth Number
The maximum x-position of a sub-sub-menu, measured from the start of the menu. A sub-sub-menu will be placed on the left side in stead of right side of its parent-item when its x-position would be greater than this point.
If "0", the remaining page-width is taken.




CSS definitions

The plugin makes a distinction between 2 types of items:

  • top-items (first level of the menu)
  • sub-items (all lower levels of the menu)

All items can be clickable or non-clickable:

  • clickable (a link): a-element
  • non-clickable (text): span-element

In the CSS a distioncion between 4 stages is made for both types of items:

  • normal item
  • normal item mouse-over
  • item of current page
  • item of current page mouse-over

This results in the following CSS definitions:

a.toplevel 						/*	normal clickable top-item */
span.toplevel					/*	normal non-clickable top-item */
a.toplevel.hover				/*	normal clickable top-item mouse-over */
span.toplevel.hover 			/*	normal non-clickable top-item mouse-over */
span.toplevel.selected			/*	current page in top-item (non-clickable) */
span.toplevel.selected.hover	/*	current page in top-item mouse-over (non-clickable) */

a.sublevel						/*	normal clickable sub-item */
span.sublevel					/*	normal non-clickable sub-item */
a.sublevel.hover				/*	normal clickable sub-item mouse-over */
span.sublevel.hover				/*	normal non-clickable sub-item mouse-over */
span.sublevel.selected			/*	current page in sub-item (non-clickable) */
span.sublevel.selected.hover	/*	current page in sub-item mouse-over (non-clickable) */

Please Note: The classes toplevel, sublevel and hover are added to the a- and span-elements by the plugin, the selected-class is not.
